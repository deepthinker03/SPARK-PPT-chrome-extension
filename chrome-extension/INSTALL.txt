🚀 SPARK PPT DOWNLOADER - STEP-BY-STEP INSTALLATION GUIDE
=========================================================

⚠️ COMMON ISSUE FIX: If you get "Manifest file is missing" error, follow these exact steps:

📁 STEP 1: PREPARE THE FOLDER
1. Download/extract the extension files to your computer
2. Make sure you have a folder named "chrome-extension" (or "Spark-PPT-Downloader")
3. Open the folder - you should see these files:
   ✅ manifest.json
   ✅ popup.html
   ✅ popup.js
   ✅ popup.css
   ✅ content.js
   ✅ content.css
   ✅ background.js
   ✅ icons/ folder (with icon files)
   ✅ libs/ folder (with library files)

🔧 STEP 2: INSTALL IN CHROME
1. Open Google Chrome browser
2. Type in address bar: chrome://extensions/
3. Press Enter
4. In the top-right corner, turn ON "Developer mode" (toggle switch)
5. You'll see new buttons appear: "Load unpacked", "Pack extension", "Update"

📦 STEP 3: LOAD THE EXTENSION
1. Click "Load unpacked" button
2. Navigate to and SELECT THE MAIN FOLDER containing manifest.json
   ⚠️ IMPORTANT: Select the folder itself, not individual files!
   ⚠️ The folder you select must contain manifest.json directly inside it
3. Click "Select Folder" (Windows) or "Open" (Mac)

✅ STEP 4: VERIFY INSTALLATION
After successful installation, you should see:
• "Spark PPT Downloader" extension card with icon
• Extension status: ON (blue toggle)
• No error messages

🎯 STEP 5: START USING
Method 1 - Auto Detection:
• Visit slideshare.net or scribd.com
• Go to any presentation page
• Look for floating "Download" button (bottom-right)
• Click and select PDF or PPT

Method 2 - Manual URL:
• Click Spark PPT icon in Chrome toolbar
• Paste presentation URL
• Select format and click "Download Now"

🚨 TROUBLESHOOTING COMMON ERRORS:

❌ "Manifest file is missing or unreadable"
✅ Fix: Make sure you're selecting the correct folder that contains manifest.json

❌ "Failed to load extension"  
✅ Fix: Check that all files are present and not corrupted

❌ "Extension doesn't appear in toolbar"
✅ Fix: Look for puzzle piece icon in toolbar, click it and pin Spark PPT

❌ "Download button doesn't appear"
✅ Fix: Refresh the presentation page and wait a few seconds

❌ "Download fails"
✅ Fix: Make sure presentation is publicly accessible

├── popup.html

├── popup.css

🔄 NEED TO UPDATE?
1. Go to chrome://extensions/
2. Find "Spark PPT Downloader"  
3. Click the refresh/reload icon
4. Changes will take effect immediately
